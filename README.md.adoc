== Ubuntu Server Installer

Github: [https://github.com/GSCloud/ubuntu_server_installer] +
BitBucket: [https://bitbucket.org/gscloud/ubuntu_server_installer]

=== Usage

* run *make* to execute common scripts
* _bin/_ ➡️ all installation scripts
* _crypt/_ ➡️ OpenSSL AES-256 encryption demo
* _dist/_ ➡️ various files for new server installation

Tested on Ubuntu 18.04+ amd64.
